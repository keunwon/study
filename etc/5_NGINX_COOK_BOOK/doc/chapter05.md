# 5. 프로그래머빌리티와 자동화

# 5.0. 소개
- 프로그래머빌리티: 프로그래밍을 통해 상호작용하는 능려
- 설정 관리 도구
    - 클라우드 시대에 매우 귀중한 유틸리티
    - 서버 설치와 설정을 자동화
    - 대규모 웹 애플리케이션으 ㄹ다루는 엔지니어는 직접 서버 설정을 만지지 않고 널리 사용되는 설정 관리 도구를
    선택해 사용할 수 있음
